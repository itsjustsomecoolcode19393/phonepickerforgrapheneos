How to install GrapheneOS

No command line needed. The official web installer handles everything in about 15 minutes. These steps work for all supported Pixel devices.

✓ No technical experience required ✓ Takes ~15 minutes ⚠ Wipes the phone — start fresh
1
Buy the right phone
You need an unlocked Pixel — never carrier-locked (no KPN, T-Mobile, Vodafone branding). Buy direct from the Google Store, Bol.com, or an unlocked retailer. Carrier models often have a permanently fused bootloader.
2
Enable Developer Options
On first boot, skip Google sign-in and connect to Wi-Fi. Then go to Settings → About phone and tap Build number 7 times until you see "You are now a developer." Then go to Settings → System → Developer options and enable OEM unlocking.
3
Open the web installer on a desktop
On your computer, open grapheneos.org/install/web in Chrome or Edge — not Firefox (WebUSB incompatibility). Connect the phone via USB. Use the original cable or a quality USB-C cable — cheap cables cause failures.
4
Unlock the bootloader
The installer guides you through rebooting into fastboot mode. On the phone, use volume buttons to navigate and power to confirm. Accept the "Unlock bootloader?" prompt — this wipes all data. That's fine, the phone is new.
5
Flash GrapheneOS
The installer downloads the correct GrapheneOS build for your exact device and flashes it automatically. Takes 3–5 minutes. Do not unplug the phone during this step. The installer shows clear progress and will tell you when it's done.
6
Re-lock the bootloader Critical
After flashing, the installer prompts you to lock the bootloader again. Always do this. Re-locking restores Verified Boot — which cryptographically checks the OS on every startup and alerts you if anything has been tampered with. This is what makes GrapheneOS more secure than other custom ROMs.
7
First GrapheneOS setup
The phone boots into GrapheneOS. Go through setup — no Google account needed at any point. Set a strong PIN or passphrase (avoid patterns). When asked about location services, choose the GrapheneOS geocoding service — not Google's.
8
Disable OEM unlocking again
Go to Settings → System → Developer options and turn off OEM unlocking. This prevents anyone with physical access to your phone from being able to unlock the bootloader and tamper with the OS — even if they know your PIN.
💡
Something went wrong?
The web installer has clear error messages and a recovery path for every failure mode. Nothing during installation can permanently damage the phone. The community forum at discuss.grapheneos.org is also very responsive.
🏦
Need your banking app?
Create a separate user profile (Settings → System → Multiple users) and install Sandboxed Google Play there only. Your main profile stays completely Google-free. Most major banking apps work fine this way.
🔒
Key security settings after setup
Enable PIN scramble, set a VPN as always-on, configure auto-reboot (72h), and set USB-C to charge-only when locked. All found in Settings → Security.
