Why GrapheneOS?

GrapheneOS is a privacy and security focused Android OS built on top of AOSP — the open-source Android base. Here's what makes it different.

🔒
No Google by default
Ships with zero Google services. No Play Store, no Firebase, no tracking. You choose what to install and what data leaves your device.
🛡️
Hardened security
Hardware memory tagging, hardened malloc, exploit mitigations, Verified Boot, and a sandboxed app model that goes far beyond stock Android.
📱
Still a great Android
Full Android compatibility. Works with most apps. You can optionally add Sandboxed Google Play in an isolated profile for apps that need it, like banking.
🔑
Per-app permissions
GrapheneOS adds network, sensors, storage scope, and contact scope permissions that stock Android doesn't have. You control exactly what each app can access.
👥
Multiple profiles
Completely isolated user profiles — one for daily use with no Google, one for banking with Sandboxed Play. Apps in different profiles can't see each other.
🔄
Fast security updates
GrapheneOS applies Android Security Bulletin patches within days of release — often faster than stock Pixel. Active development with a dedicated security team.

How to install GrapheneOS

No command line needed. The official web installer handles everything in about 15 minutes on a fast connection.

1
Buy an unlocked Pixel
Never carrier-locked (KPN, T-Mobile, Vodafone). Buy direct from Google Store or an unlocked retailer. Carrier models often have a permanently fused bootloader.
2
Enable Developer Options
Skip Google sign-in on first boot. Go to Settings → About phone → tap Build number 7 times. Then Settings → System → Developer options → enable OEM unlocking.
3
Open the web installer on a desktop
Go to grapheneos.org/install/web in Chrome or Edge — not Firefox (WebUSB incompatibility). Connect phone via USB with a quality cable.
4
Unlock the bootloader
The installer guides you into fastboot mode. Accept the "Unlock bootloader?" prompt on the phone. This wipes all data — that's expected.
5
Flash GrapheneOS
The installer downloads and flashes the correct build for your exact device automatically. Takes 3–5 minutes. Do not unplug the phone during this step.
6
Re-lock the bootloader Critical
Always do this. Re-locking restores Verified Boot — which cryptographically checks the OS on every startup. This is what makes GrapheneOS more secure than other custom ROMs.
7
Complete setup
No Google account needed at any point. Set a strong PIN or passphrase. Choose GrapheneOS geocoding — not Google's — for location services.
8
Disable OEM unlocking again
Settings → System → Developer options → turn off OEM unlocking. Prevents physical attackers from reflashing the device even if they know your PIN.
💡
Something went wrong?
The installer has clear error messages and recovery paths for every failure. The community forum at discuss.grapheneos.org is very responsive.
🏦
Need banking apps?
Create a separate user profile (Settings → System → Multiple users) and install Sandboxed Google Play there only. Most banking apps work fine this way.
🔒
After setup
Enable PIN scramble, set Proton VPN as always-on with kill switch, configure auto-reboot timer (72h), and set USB-C to charge-only when locked. All in Settings → Security.
